‚ùì The Challenge

  • Data Scarcity: "Normal" X-rays are abundant, but disease data is rare and expensive.
  • "Black Swan" Failure: Standard AI misses novel pathologies it hasn't seen before.
  • No Explainability: Doctors don't trust scores. They need to see where the problem is.

üí° The Solution: One-Class Learning

Train only on Healthy Anatomy.

  1. Model learns the "Manifold of Health".
  2. It reconstructs healthy lungs perfectly.
  3. When it sees a disease, it fails to reconstruct it.
  4. Input - Output = Difference Map (The Lesion)
