‚ùì The Challenge

  • Data Scarcity: "Normal" X-rays are abundant, but disease data is rare and expensive to label.
  • The "Black Swan" Failure: Standard AI classifiers only find diseases they were trained on. They miss novel pathologies.
  • Lack of Explainability: Doctors don't trust "Black Box" scores. They need to see where the problem is.

üí° The Solution: One-Class Learning

Instead of training on diseases, we train only on Healthy Anatomy.

  1. Model learns the "Manifold of Health".
  2. It reconstructs healthy lungs perfectly.
  3. When it sees a disease, it fails to reconstruct it.
  4. Input - Output = Difference Map (The Lesion)

Model Mechanism Pros Cons

ViT Attention Global context awareness. Data hungry; blocky artifacts.
